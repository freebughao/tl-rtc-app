### 准备

1. 安装nodejs环境

2. 安装mysql环境

3. 安装redis环境

### 安装步骤

1. 克隆仓库：

    ```bash
    git clone https://github.com/tl-open-source/tl-rtc-app.git
    ```
   
2. 进入项目目录：

    ```bash
    cd tl-rtc-app
    ```
   
3. 安装依赖：

    ```bash
    npm install
    ```

### 使用说明
1. 启动应用 - Api

    ```bash
    npm run http-api

    或者

    npm run https-api
    ```

2. 启动应用 - Socket

    ```bash
    npm run http-socket

    或者

    npm run https-socket
    ```
   
3. 打开浏览器并访问 `http://localhost:9096`。
